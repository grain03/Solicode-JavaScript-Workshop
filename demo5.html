Recherche et suppression dans une chaine

Enoncé

Ecrire un programme Javascript qui demande à l’utilisateur de saisir une phrase dans une zone de texte et un mot dans une autre zone de texte. Le clique sur le bouton rechercher affiche un message si le mot existe dans la phrase ou non. Le clique sur le bouton supprimer permet d’effacer toutes les occurrence du mot dans la phrase ( si le mot existe bien sûr).

Démo



Le message de recherche s’affiche ici Ou la nouvelle phrase après la suppression des occurences

Code source

function btn1() {

document.getElementById("result1").style.display = "block";
var inputedvoyelles = document.getElementById('voyelles').value;
inputedvoyelles = inputedvoyelles.toLowerCase();
let count = 0;

for(i = 0; i < inputedvoyelles.length; i++) {

if(inputedvoyelles.charAt(i) == 'a' || inputedvoyelles.charAt(i) == 'e' || inputedvoyelles.charAt(i) == 'y'|| inputedvoyelles.charAt(i) == 'i' || inputedvoyelles.charAt(i) == 'o' || inputedvoyelles.charAt(i) == 'u') {
count++;
}

}
document.getElementById('result').innerHTML = "La phrase que vous avez saisi contient " + count + " voyelles";
}
